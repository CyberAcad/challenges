# Follow the Data

**Category:** Intro to Cybersecurity
**Difficulty:** Easy
**Points:** 50

> This challenge is already deployed on CTFd. See `files/` for the trace log given to students.

## Description

A simplified trace of a login event has been captured on CyberAcad's servers! At which point within the logs does the IPC loop reach "Storage"? Submit the timestamp of the event.

**Flag format:** `cyber{MM/DD/YY HH:MM:SS}`
