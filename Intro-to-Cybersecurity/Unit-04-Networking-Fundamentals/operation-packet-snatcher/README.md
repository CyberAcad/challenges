# Operation Packet Snatcher

**Category:** Intro to Cybersecurity  
**Difficulty:** Hard  
**Points:** 300

## Description

🕵️ CLASSIFIED — EYES ONLY 🕵️

You're a junior analyst at a (totally fictional) cyber intelligence agency. Your team intercepted a secret message being transmitted across a covert network — but there's a problem.

The message was **transformed at every network hop** before reaching its destination. By the time you intercepted it at the final node, it looks like complete gibberish.

Luckily, your team was able to map the network path and figure out **exactly what each node did** to the message as it passed through.

Your mission: **work backwards** from the intercepted gibberish to recover the original secret message.

Open `mission_briefing.txt` — it has everything you need.

**Flag format:** `cyber{decoded_message}`
