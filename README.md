# 🛡️ CyberAcad — Challenges

Hands-on cybersecurity challenges for students and aspiring security professionals, created by [CyberAcad](https://github.com/CyberAcad).

## 📁 Structure

Challenges are organized by course/category:

```
challenges/
├── Intro-to-Cybersecurity/
│   ├── challenge-name/
│   │   ├── README.md        # Description, hints, difficulty
│   │   ├── flag.txt         # The flag (keep private or use .gitignore)
│   │   └── files/           # Any files provided to participants
│   └── ...
└── (future categories)
```

## 🎯 Categories

| Category | Description |
|---|---|
| **Intro to Cybersecurity** | Beginner-friendly challenges covering security fundamentals |

## 🚀 Getting Started

1. Pick a category folder
2. Read the challenge `README.md`
3. Solve the challenge and capture the flag!

## 📝 Flag Format

```
CyberAcad{flag_text_here}
```
